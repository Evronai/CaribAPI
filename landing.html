API Examples

Integrate Caribbean business data in minutes with our simple REST API:

// Get all businesses in Port of Spain GET /api/v1/businesses/?city=Port%20of%20Spain

// Search for businesses GET /api/v1/businesses/search?q=brewery

// Get business by registration number GET /api/v1/businesses/registration/TRN20240001

// Authentication header X-API-Key: your_api_key_here

Full documentation available at /docs endpoint when API is running.
